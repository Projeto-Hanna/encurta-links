## Encurta links - Projeto Hanna
Status: em desenvolvimento

### Descrição

Projeito feito para auxiliar em redirecionamento e encurtamento de links.

### Setup

```bash
$ yarn install
```

### Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

### Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
